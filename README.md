## 编程练习2
### 使用基于ReLU的神经网络来拟合一个有界闭集函数

我同样在 `chapter4_ex2` 文件夹中实现了 `纯numpy` 和 `numpy + pytorch` 两个版本，为加快训练速度，`pytorch` 使用了GPU版本。
其中，`function_numpy.ipynb` 中是 `纯numpy` 版本的实现，可以达到 0.0428 的 MSE；`function_pytorch.ipynb` 中是 `numpy + pytorch` 版本的实现，可以达到 0.0018 的MSE。

以下的分析基于 `numpy` 版本给出。

#### 函数定义

为了验证两层 ReLU 神经网络的通用近似能力，我们选取了一个包含局部振荡与全局非线性趋势的复合函数作为目标拟合函数。
具体数学表达式如下：
$$f(x) = \sin(3\pi x) + 0.5x^2$$
其中，自变量的定义域限定为 $x \in [-1, 1]$。

该函数结合了三角函数的周期性波动和二次函数的多项式增长，具备多个极值点，能够充分检验模型对复杂非线性映射的学习能力。

#### 数据采集

我们采用均匀采样与随机打乱相结合的方式构建数据集：

数据生成：在区间 $[-1, 1]$ 内均匀生成 2000 个 $x$ 样本点，并根据上述函数定义计算对应的真实 $y$ 值。

数据集划分：将生成的数据集随机打乱后，按照 80% / 20% 的比例划分为训练集和测试集。

形状说明： 输入数据维度为 (N, 1)，输出标签维度为 (N, 1)。由于该函数的输出值域在一个合理的范围内（大致在 $[-1, 1.5]$ 之间），不需要对目标变量 $y$ 进行额外的归一化处理。

#### 模型描述

我们构建了一个标准的两层前馈神经网络（包含一个隐藏层和一个输出层）：

输入层： 接收 1 维输入特征 $x$。

隐藏层： 包含 128 个神经元。该层采用 ReLU 作为激活函数，即 $a = \max(0, z)$。
大量的 ReLU 神经元协同工作，为网络提供了将输入空间划分并进行分段线性映射的能力。

输出层： 包含 1 个神经元，不使用激活函数（即线性输出），直接输出预测的连续值 $\hat{y}$。

损失函数与优化： 模型采用均方误差 (MSE) 作为损失函数，衡量预测值与真实值之间的差异。

#### 拟合效果
模型在训练了 20000 个 Epoch 后，在训练集和测试集上均表现出了优异的拟合能力。

定量分析：

最终训练集 MSE 损失：0.0428

最终测试集 MSE 损失：0.0598

测试集损失与训练集损失非常接近，表明模型没有发生明显的过拟合，具有良好的泛化能力。

即使是仅含单隐藏层的浅层网络，只要配备了非线性激活函数（如 ReLU）且神经元数量足够，就能以极高的精度逼近复杂的连续函数。
